Phaedra-Shell is a reverse tcp-shell that allows remote code excution in windows environment targets.
The server listens for incoming connections and can send commands to the client, which executes them and returns the output. Designed for educational purposes, this project demonstrates basic socket programming and multithreading in Python.

## features 
- `Remote Command Execution`: Send commands from the server to the client.
- `System Information`: Automatically sends system details upon connection.
- `Multithreading`: Handles multiple client connections simultaneously.
